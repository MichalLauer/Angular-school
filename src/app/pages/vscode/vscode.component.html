Něco málo o Tvém budoucím IDE

  • Jedná se o vývojové prostředí, které se hodí pro vývoj jednodušších aplikací(především webových).
  • VS Code je jednoduchý a multiplatformní (spustitelný na macOS, Linux a Windows).
  • VS Code nemá žádný vlastní formát projektů. (projekty se pouze nahrávají)
  • Hlavní přednosti VS Code:
  • IntelliSense - podpora doplňování kódu.
  • Debugging – testotvání a oprava chyb přímo v editoru.
  • Git commands built-in – pro usnadnění práce s rozsáhlými projekty přímo v editoru.
  • Extensible and customizable – podporování vývoje řadou stažitelných doplňků.
  • Stáhneš ho přímo z ofiko stránek Microsoftu

Popis základního panelu

  • File – otevírání, zavírání, ukládání a nastavení oken, složek, souborů
  • Edit – základní funkce (kopírování, vkládání, nahrazování, komentáře, … + funkce emmet
  • Selection – výběr, kopírování, posouvání celých bloků kódu a více kurzorová funkce
  • View – nastavení zobrazení veškerého rozhrání VS Code (přidávání editorů, přepínání zobrazení, …)
  • Go – funkce pro pohybování se ve složkách, editorech a kódu
  • Debug – ovládání ladícího programu
  • Terminal – ovládání terminálu
  • Help – o programu, licence programu, tipy a triky, … (detailní popis všeho ve VS Code a kolem něj)

Panel aktivit

  • Průzkumník souborů (Ctrl+Shift+E) – nahrávání projektů, vkládání nových složek a souborů
  • Vyhledávání v souborech (Ctrl+Shift+F)
  • Správa zdrojového kódu (Ctrl+Shift+G)
  • Ladění kódu (Ctrl+Shift+D) – zde můžeme náš kód spustit a vyhledávat chyby
  • Správce rozšíření (Ctrl+Shift+X) – zde nalezneme veškeré doplňky týkající se VS Code

Panel nastavení

  • Command Palette – otevře příkazovou řádku, pro zadání editorových příkazů
  • Settings – zde můžeme nastavit text editoru, pracovní prostředí a jeho vlastnosti, vlastnosti oken, samotnou aplikaci (proxy, updaty, dálkový přenos dat) a nastavení rozšíření, dělí se na User Settings(týká se jakékoli instance VS Code - globální nastavení) a Workspace Settings(pouze užívaný VS Code)
  • Keyboard Shortcuts – nastavení klávesových zkratek
  • User Snippet – otevře řádek se Snippety
  • Color Theme – nastavení barevného vzhledu prosředí
  • File Icon Them – nastavení vzhledu složkových ikon
