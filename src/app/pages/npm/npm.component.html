Nejjednoduší práce s Angularem

Pokud chceme co nejefektivnějí a nejjeddouše vyvýjet webové aplikace v Angularu, je třeba si nainstalovat Node Package Manager (NPM) a Angular CLI. V této kapitole si přiblížíme co je co a ukážeme základní příkazy.

Správce balíčků

NPM je správce balíčků. Přes něj dokážete pomocí jednoho příkazu nainstalovat mnoho užitečných a nesmyslných rozšíření pro Vaše potřeby. Po nainstalovaní se budeme pohybovat čistě v konzoli. Níže je krátký seznam příkazů, které by se mohli hodit.

  • npm install XXX - nainstaluje požadovaný package (po přidání -g se nainstaluje do celého pc)
  • npm update - aktualizuje všechny baličky
  • npm list - zobrazí list všech nainstalovaných balíčků

Všechno o tom CLI

Pokud máme nainstalovaný NPM, je na čase se pustit do práce s Angularem. Na to použijeme následující příkaz:

npm install -g @angular/cli

Po nainstalování je práce s konzolí velice snadná a nemusíme se starat o složitější operace a vkládání importů do .ts souborů. Níže najdete podrobnější seznam všech důležitých příkazů.

ng new project-name
 Založí nový Angular projekt, se kterým je možné hned pracovat a developerovat.
ng generate component componentName
 CLI nám vygeneruje nový komponent a obstará za nás všechny závislosti a importy.
ng serve -o
 Tento příkaz nám založí lokální server (localhost:4200), na kterém pobeží naše stránka. Parametr -o nám rovnou otevře stránku v prohlížeči.
