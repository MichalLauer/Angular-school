Práce s daty v Angularu

Dvojitý závorky jako vnoření

První věc kterou se naučíme, je vlkádání (neboli interpolace) textu přímo do HTML. Pokud máme v TypeScriptu uloženou nějakou hodnotu, kterou lze zkonvertovat, tak ji můžeme ukazovat na HTML stránce přímo pomocí {{{{interpolation_string}}}} Níže uvidíte názorný příklad

... HTML stránce přímo pomocí { { interpolation_string } } ...
public interpolation_string: String = 'proměnná';

Bindování proměnné

Tak jak můžeme ukazovat informace na stránce, tak můžeme nastavovat informace objektům z dokumentu. pomocí syntaxe [vlastnost]="ts" se dají měnít vlastností objektů.

<input id="div-color" type="text"
[style.backgroundColor]="this.input_backgroundColor"
[value]="this.input_backgroundColor">
...
<button> (click)="changeColor()">Change color!</button>

Event Binding

Díky Angularu můžeme elegantně reagovat na události prvku. Syntaxe je velmi jednoduchá - názvy eventů jsou stejné jako v Javascriptu, akorát se odstraní předpona "on". Takže z onclick se stane click, onmouseover se stane mouseover atp. Toto klíčové slovo se zabalí do kulatých závorek a za rovnítko se píše Typescript.


placeholder div
<button> (click)="updateEventColor('red');">Červená</button>
<button> (click)="updateEventColor('green');">Zelená</button>
...
(<HTMLInputElement>document.getElementById('event-div')).style.backgroundColor = color;

Two-way data binding

Použítím jednoduchého zápisu lze data ukládat vždy, pokud se na HTML dokumentu změní nebo aktualizují. Používá se k tomu direktiva [(ngModel)] oplácaná kulatými a hranatými závorkami. za rovnítko dáme název proměnné, kam se má hodnota uložit.

{{ngModelVar}}

<input> type="text" [(ngModel)]="this.ngModelVar">
<p> { { ngModelVar } } </p>
