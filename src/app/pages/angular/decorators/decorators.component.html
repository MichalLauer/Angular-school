Dekorátory

Dekorátory slouží k upravení třídy. Mohou se do nich ukládat jak metada, tak se dají použít jako vstupní a výstupní metody ke komponentu. Dekorovat komponent můžeme mnoha způsoby a patří to spíše k pokročilejšímu vědění, proto se naučíme "pouze" posílání dat, vystřelování eventů a reagovaní na eventy

@Input

@Input nám umožní dat komponentu "vlastnosti", které můžeme poté z venku nastavit - Podobně jako bychom nastavili např. src u obrázku. V komponentu poté lze nastavit alias dané proměnné.

<input id="input-var" type="text" [(ngModel)]="this.inputVarHolder">
<input id="input-alias" type="text" [(ngModel)]="this.inputAliasHolder">
Zavolání komponentu
<app-input [inputVar]="this.inputVarHolder" [aliasInput]="this.inputAliasHolder"></app-input>
Input v komponentu
@Input() inputVar: String;
@Input('aliasInput') aliasVarString: String;

@Output

Díky tomuto dekorátoru můžeme odesílat data z komponentu. Při reagování na vlastní událost můžeme zase použít název objektu, nebo alias. Zároveň pomocí $event můžeme z komponentu převzít data, která nám poslal.

@Output() var = new EventEmitter<String>();
this.outputVar.emit(this.value);
...
@Output('alias') outputVar = new EventEmitter<String>();
this.var.emit(this.value);
...
<app-output (var)="reactOnVar()" (alias)="reactOnAlias($event)"></app-output>

@HostListener

Tento dekorátor nám umožní reagoat na vnější události uvnitř Typescript souboru. Jednoduše zaaliasujeme název eventu a objektu přiřadíme metodu.

@HostListener('mouseover') onHover() {
    document.getElementById('text').style.color = 'red';
}
@HostListener('mouseout') onOut() {
    document.getElementById('text').style.color = 'black';
}
...
<app-listener></app-listener>
